## Project: Dog Walk Analytics

A Python data analytics project analyzing dog walk data for two dogs (Bussi & Mila).

### Files

- `dogs_simple.py` — Raw walk data (minutes per walk) and temperature data, with daily aggregations
- `dogs_walks_ai.py` — Linear regression model (slope + intercept) to predict walk duration from temperature
- `dogs_comparison.py` — Side-by-side stats comparing Bussi vs Mila

### Conventions

- Standard Python 3
- No external dependencies beyond standard library
- Variable names are descriptive (sometimes playful)
- Print-based output (no logging framework)
